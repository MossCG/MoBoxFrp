# 后端指令目录
> `[]` 为必填参数 <br>
> `<>` 为选填参数 <br>
> `前缀` 为指令最开始的词，多前缀可对应同一指令 <br>

## 退出后端
> 可用前缀：exit、quit <br>
> 用途：退出后端（其实你直接关也是没问题的） <br>
> 格式：`[前缀]`

| 参数  | 类型  | 参考  | 详解  |
|-----|-----|-----|-----|
| 无参数 | -   | -   | -   |

## Debug开关
> 可用前缀：debug、api <br>
> 用途：开关debug信息显示（启动时默认关闭） <br>
> 格式：`[前缀]`

| 参数  | 类型  | 参考  | 详解  |
|-----|-----|-----|-----|
| 无参数 | -   | -   | -   |
