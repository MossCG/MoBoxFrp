# 后端权限文档

## 权限分级
> 不同权限有不同的权限数字等级 <br>
> 数字越大，权限越高

> 预留了很大的空间给VIP权限组插件分配

| 名称    | 英文ID     | 数字等级 | 注释              |
|-------|----------|------|-----------------|
| 站长    | admin    | 999  | 最高权限所有者         |
| 管理    | staff    | 800  | 管理员权限           |
| 节点所有者 | provider | 600  | 提供节点的用户         |
| 普通用户  | default  | 200  | 默认用户组           |
| 封禁用户  | banned   | 100  | 被封禁的用户，基本上啥权限没有 |
